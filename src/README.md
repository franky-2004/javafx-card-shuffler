# JavaFX Card Shuffler

A simple JavaFX program that shuffles cards and displays messages when two cards match.

## Features
- Card shuffling
- "Match found!" or "No match" display

## How to Run
- Open in IntelliJ or Eclipse
- Run `Main.java`
- Make sure JavaFX is set up

## Tech
- Java
- JavaFX
